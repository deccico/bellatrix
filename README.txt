===========
Bellatrix
===========

Bellatrix is a comprehensive set of tools to automate the management of Amazon EC2 services.

    * bellatrix bewitch_ami - Starts a new instance, applies a configuration to it and finally saves it into a new ami.
    * bellatrix copy2s3 - Copy a file or directory to a S3 bucket.
    * bellatrix burn - Burns a running instance into a new ami.
    * bellatrix perm2ami - Gives permissions to other accounts to launch your ami.
    * bellatrix startinstance - Starts a new EC2 instance.
    
For more information just execute *bellatrix -h* or *bellatrix command -h*
